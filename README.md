# Accepted - Сайт компании по обслуживанию инженерных сетей

Статический сайт для компании Accepted. Полностью адаптивный, без внешних зависимостей.

## Структура проекта

```
acceptedgroup/
├── index.html          # Главная страница
├── projects.html       # Страница проектов
├── contacts.html       # Страница контактов
├── style.css           # Стили
├── script.js           # JavaScript
├── send.php            # Обработчик формы обратной связи
├── .htaccess           # Настройки Apache
├── deploy.sh           # Скрипт для деплоя на хостинг
└── assets/             # Ресурсы
    └── images/
        ├── logo-black.svg
        ├── logo-white.svg
        └── logo-accepted.png
```

## Быстрый старт

1. Откройте `index.html` в браузере
2. Или загрузите на хостинг через FTP

## Деплой

Используйте `deploy.sh` для автоматического деплоя на хостинг (настройте FTP данные в файле).

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript
- PHP (для формы обратной связи)
